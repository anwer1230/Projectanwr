# نظام مراقبة التليجرام المتقدم

## نظرة عامة
نظام مراقبة تليجرام متقدم مصمم لمركز الخدمات الطلابية والأكاديمية. يسمح للمستخدمين بربط حساباتهم على التليجرام لمراقبة الرسائل تلقائياً وإرسال ردود آلية بناءً على الكلمات المفتاحية وإدارة جلسات تليجرام متعددة عبر واجهة ويب.

## تفضيلات المستخدم
- أسلوب التواصل المفضل: لغة بسيطة وواضحة
- اللغة: العربية مع دعم RTL

## بنية النظام
### الواجهة الأمامية
- **التقنيات**: HTML5, Bootstrap 5, JavaScript مع Socket.IO
- **النمط**: Progressive Web App (PWA) مع دعم العربية RTL وإمكانيات العمل دون اتصال
- **التواصل الفوري**: واجهة WebSocket باستخدام Socket.IO للتحديثات المباشرة والإشعارات
- **مكونات الواجهة**: مكونات Bootstrap مع CSS مخصص وتوطين عربي
- **إدارة الحالة**: JavaScript للعميل مع تخزين محلي للإعدادات

### الواجهة الخلفية
- **الإطار**: Flask مع Flask-SocketIO لدعم WebSocket
- **إدارة الجلسات**: تخزين الجلسات في ملفات في مجلد `/sessions`
- **المصادقة**: مصادقة إدارية بسيطة بكلمة مرور مع إدارة جلسات Flask
- **نموذج التزامن**: نهج قائم على الخيوط مع أقفال آمنة للخيوط
- **معالجة الرسائل**: بنية مدفوعة بالأحداث باستخدام مكتبة Telethon
- **نظام التنبيهات**: نظام إدارة تنبيهات قائم على قائمة الانتظار مع إشعارات فورية

## التبعيات الخارجية
### المكتبات الأساسية
- **Flask**: إطار ويب للخادم HTTP والتوجيه
- **Flask-SocketIO**: تطبيق WebSocket للتواصل الفوري ثنائي الاتجاه
- **Telethon**: مكتبة عميل تليجرام لتكامل API ومعالجة الرسائل
- **Bootstrap 5**: إطار CSS للواجهة مع دعم RTL
- **Font Awesome**: مكتبة الأيقونات للعناصر البصرية

## مكونات PWA
- **Service Worker**: إمكانيات التخزين المؤقت دون اتصال ومزامنة الخلفية
- **بيان تطبيق الويب**: تكوين PWA مع التوطين العربي وأيقونات التطبيق
- **إشعارات الدفع**: دعم إشعارات المتصفح للتنبيهات الفورية
